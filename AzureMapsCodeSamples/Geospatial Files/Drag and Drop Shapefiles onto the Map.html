Drag and Drop Shapefiles onto the Map This sample shows how load add support for dragging and dropping zipped shapefile (.shp, .dbf, .prj) files from your local computer file system on to the map and having them render on the map. This sample uses the open source Shapefile.js library which converts Shapefiles into GeoJSON which can easily be consumed by the map control. It also makes use of the open source Catiline.js library which makes working with web workers easier.
